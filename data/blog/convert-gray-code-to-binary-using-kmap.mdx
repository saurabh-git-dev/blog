#### Truth Table -

To find truth table first it is neccessary to know about gray code and binary code then it can be
found easily.

#### How To Solve -

Now, To solve it we need truth table which i have already made.

Lets consider first for output B<sub>3</sub> , see column B<sub>3</sub> and find the rows
where B<sub>3</sub> is 1 and notedown corresponding input values.

Now, for this corresponding value write decimal value.

Now, once you got all of these things you are ready for writing your first expression -

write it as , B<sub>3</sub> = Σ<sub>m</sub>( 8, 9, 10, 11, 12, 13, 14, 15 )

Similarly take output B<sub>2</sub> and find 1's in the column and for input find corresponding
decimal value and write it.
